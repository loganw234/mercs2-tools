MERCENARIES 2 MODDING HUB & UTILITIES

🗃️

SAVE GAME EDITOR

Modify PC save game profiles (.profile). Adjust cash, fuel, faction relations, and manage individual stockpiles of all 134 support items. Features a raw database modifier with safety syntax checks. Runs completely client-side in your browser.

LAUNCH SAVE EDITOR
📜

LUA MENU BUILDER

A web tool to build structured Lua menu script definitions for the in-game cheat menu system. Design multi-page custom options pipelines with a built-in game screen simulator.

LAUNCH MENU BUILDER
🎯

MISSIONFORGE

Web companion to the in-game contract authoring tool. Paste a MISSIONFORGE_EXPORT dump, refine objectives and mission details, then copy out a ready Contract.Register{} definition for the custom contract framework.

LAUNCH MISSIONFORGE
🦴

MODEL RIG BROWSER

Every bone and hardpoint on all 329 models in the game (characters, vehicles, weapons), searchable and viewable as a live 2D skeleton. Recovered via hash-cracking the same string table behind the save editor's item names — 64% of all 16,690 bone slots have a confirmed real name so far, the rest reachable through a synthetic handle.

LAUNCH RIG BROWSER
🛠️

MERCENARIES 2 MODKIT

A desktop mod manager for Mercenaries 2. Load mods, detect conflicts, resolve asset collisions, and package validated WAD archives (vz-patch.wad) without using the command line.

VIEW ON GITHUB
📖

CHEAT MENU & MOD GUIDE

A very well-made and thorough guide detailing the setup process for manual cheat menus, developer scripts, and config modding.

VIEW ON GITHUB

SECUROM BYPASS

Removes the legacy SecuROM DRM protection from the game executable and alters it to load the community's mod loader library.
